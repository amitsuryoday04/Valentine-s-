# ❤️ Valentine's Website

A beautiful Valentine's Proposal Website made using HTML, CSS and JavaScript.

##   Link
- https://amitsuryoday04.github.io/Valentine-s-/

## ✨ Features

- ❤️ Romantic UI Design
- 💕 "Do You Love Me?" Question
- ✅ Yes Button
- ❌ No Button runs away when hovered
- 🎵 Background Music Support
- 🖼️ Personal Photo
- 💖 Floating Hearts Animation
- 📱 Mobile Responsive

## 📁 Project Structure

```
Valentine-Website/
│── index.html
│── style.css
│── script.js
│── music.mp3
└── images/
    └── photo1.jpg
```

## 🚀 How to Run

1. Download or Clone this repository.
2. Add your photo to `images/photo1.jpg`.
3. Add your music as `music.mp3`.
4. Open `index.html` in your browser.

## 🌐 Deploy on GitHub Pages

1. Upload all files to GitHub.
2. Go to **Settings** → **Pages**.
3. Select **Deploy from a branch**.
4. Choose **main** branch and **/root** folder.
5. Click **Save**.

Your website will be live in a few minutes.
